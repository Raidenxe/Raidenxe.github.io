使用命令行查看Linux基本信息

发表于2022-07-19|更新于2022-07-19|Linux学习笔记
|字数总计:618|阅读时长:2分钟

命令行(CLI)是用来和计算机进行交流沟通的非常有效的方式，正像人类社会使用文字互通信息一样。

“图形用户界面让简单的任务更容易完成， 而命令行界面使完成复杂的任务成为可能”。

以下是我整理的一些查看Linux基本信息的命令行。

一、查看系统信息

查看主机名：
hostname

查看cpu信息：
lscpu

查看发行版本：
cat /etc/issue

lsb_release -a

cat /etc/redhat-release（rpm系）

查看系统信息：
uname -a

查看Linux内核版本：
uname -r

cat /proc/version

查看操作系统的位版本：
uname -m

查看设备：
ls /dev

列出所有PCI设备：
lspci -tv

列出所有USB设备：
lsusb -tv

列出加载的内核模块：
lsmod

查看环境变量：
env

二、查看资源信息

查看内存：
// -g表示单位是G，-m 表示单位是m，要使用-m，因为-g会向下取整，不准确

free -m

查看所有进程：
ps -elf

查看磁盘使用情况：
df -h

查看磁盘IO使用情况：
// 1 表示没1s刷新一下

iostat -x 1

查看所有交换分区：
swapon -s

查看系统运行时间、用户数、负载：
uptime

查看系统负载：
cat /proc/loadavg

查看cpu率等信息，类似于Windows的任务管理器：

// htop是top的升级版，可以查看更多的信息以及提供更好的可视化效果，更为直观地展示各类资源占用的情况，而glances功能相比htop更丰富，可以理解为htop 代替 top，glances 代替 htop。

top

htop

glances

三、查看用户信息

查看活动用户：
w

查看用户登录日志：
last

查看历史命令：
history

##记录的是前一次登陆以前所执行过的指令

cat .bash_history

查看用户信息：
id

查看系统所有用户：
cut -d: -f1 /etc/passwd

查看系统所有组cut -d:
-f1 /etc/group

查看当前用户的计划任务：
crontab -l

四、查看网络信息

查看本机ip:
ip a

显示或配置网络设备：
##可用于查看所有网络接口的属性

ifconfig

查看路由表：
ip route

route -n

查看防火墙设置：
iptables -L

查看所有监听端口：
netstat -lntp

查看所有已经建立的连接：
netstat -antp

查看网络统计信息：
netstat -s

网络诊断工具：
ping

路由跟踪：
traceroute
